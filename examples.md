# Road Trip Planner Examples / 示例

## Example 1: Hulunbuir (Mongolia East) 8-Day Loop / 蒙东呼伦贝尔 8 日自驾环线

**User input**: Help me plan an 8-day road trip route in Mongolia East. / 帮我生成一个蒙东自驾 8 日游的路线。

**Design approach / 设计思路**：
- Start/End: Hailar (closed loop) / 起点/终点：海拉尔（闭合环线）
- Total: ~1200–1500km / 总里程约 1200-1500km
- Daily driving: 2–4 hours, relaxed pace / 每日驾驶 2-4 小时，节奏舒缓
- Classic Hulunbuir Grand Loop covering grasslands, wetlands, forests, border areas, and lakes / 经典呼伦贝尔大环线，涵盖草原、湿地、森林、边境、湖泊多种地貌

**Route / 路线节点**：

| Day | Route / 路线 | Mileage / 里程 | Duration / 时长 |
|-----|-------------|---------------|-----------------|
| Day 1 | Hailar → Morigele River → Erguna / 海拉尔 → 莫日格勒河 → 额尔古纳 | 154km | 2.0h |
| Day 2 | Erguna → Wetland → Birch Forest → Enhe / 额尔古纳 → 湿地 → 白桦林 → 恩和 | 99km | 1.7h |
| Day 3 | Enhe → Shiwei → Linjiang / 恩和 → 室韦 → 临江 | 11km | 0.3h |
| Day 4 | Linjiang → Laoyingzui → Taiping → Mordaga / 临江 → 老鹰嘴 → 太平 → 莫尔道嘎 | 38km | 0.7h |
| Day 5 | Mordaga → Aoluguya → Genhe / 莫尔道嘎 → 敖鲁古雅 → 根河 | 4km | 0.1h |
| Day 6 | Genhe → Erguna → Heishantou / 根河 → 额尔古纳 → 黑山头 | 210km | 3.0h |
| Day 7 | Heishantou → 186 Ribbon River → Manzhouli / 黑山头 → 186 彩带河 → 满洲里 | 170km | 1.8h |
| Day 8 | Manzhouli → Hulun Lake → Hailar / 满洲里 → 呼伦湖 → 海拉尔 | 186km | 2.0h |

**Core map nodes (16) / 核心地图节点（16 个）**：
Hailar Airport, Morigele River, Erguna City, Erguna Wetland, Birch Forest Scenic Area, Enhe Russian Ethnic Village, Shiwei Port, Linjiang Village, Laoyingzui, Taiping Village, Mordaga National Forest Park, Aoluguya Reindeer Tribe, Genhe City, Heishantou Town, Manzhouli National Gate, Hulun Lake

**Output / 输出结果**：
- Personal map QR code (16-node closed loop, Hailar at both start and end) / 个人地图小程序二维码（16 个节点闭合环线）
- Markdown itinerary guide (with gas station and hospital POI search along route) / Markdown 行程指南

**Special notes / 特别提示**：
- Some sections (e.g., Linjiang to Mordaga) have weak cell signal — download offline maps in advance / 蒙东地区部分路段（如临江至莫尔道嘎）手机信号较弱，建议提前下载离线地图
- August peak season — book accommodation in Erguna, Shiwei, and Heishantou early / 8 月旺季住宿紧张

## Example 2: Qinghai-Gansu Grand Loop 8-Day Road Trip / 青甘大环线 8 日自驾

**User input**: Plan a Qinghai-Gansu Grand Loop road trip starting from Xining. Consider daily driving time and avoid fatigued driving. / 给我规划一个西宁出发的青甘大环线自驾行程，记得考虑每日自驾时长，不要疲劳驾驶。

**Design approach / 设计思路**：
- Start/End: Xining (loop) / 起点/终点：西宁（环线）
- Total: ~2000–2400km / 总里程约 2000-2400km
- Daily driving: 4–6 hours；Day 5 is a Dunhuang city rest day / 每日驾驶 4-6 小时，Day 5 敦煌市区纯玩驾驶极少

**Route / 路线节点**：

| Day | Route / 路线 | Mileage / 里程 | Duration / 时长 |
|-----|-------------|---------------|-----------------|
| Day 1 | Xining → Ta'er Monastery → Qinghai Lake / 西宁 → 塔尔寺 → 青海湖 | 276km | 4.7h |
| Day 2 | Qinghai Lake → Chaka Salt Lake → Delingha / 青海湖 → 茶卡盐湖 → 德令哈 | 370km | 4.7h |
| Day 3 | Delingha → Emerald Lake → Dachaidan / 德令哈 → 翡翠湖 → 大柴旦 | 206km | 2.4h |
| Day 4 | Dachaidan → Nanbaxian → Dangjin Pass → Aksay / 大柴旦 → 南八仙 → 当金山 → 阿克塞 | 265km | 3.3h |
| Day 5 | Aksay → Dunhuang → Mingsha Mountain / 阿克塞 → 敦煌 → 鸣沙山月牙泉 | 85km | 1.7h |
| Day 6 | Dunhuang → Mogao Caves → Jiayuguan / 敦煌 → 莫高窟 → 嘉峪关 | 409km | 4.6h |
| Day 7 | Jiayuguan → Rainbow Danxia → Zhangye / 嘉峪关 → 七彩丹霞 → 张掖 | 263km | 4.4h |
| Day 8 | Zhangye → Qilian Grassland → Menyuan → Xining / 张掖 → 祁连草原 → 门源 → 西宁 | 509km | 6.5h |

**Core map nodes (16) / 核心地图节点（16 个）**：
Xining City, Ta'er Monastery, Qinghai Lake, Chaka Salt Lake, Delingha City, Dachaidan Emerald Lake, Dachaidan Town, Aksay Kazakh Autonomous County, Dunhuang City, Mingsha Mountain Crescent Spring, Mogao Caves, Jiayuguan City, Jiayuguan Fortress, Zhangye Rainbow Danxia, Qilian Mountain Grassland, Menyuan

**Output / 输出结果**：
- Personal map QR code (closed loop)
- Markdown itinerary guide (with altitude sickness, Mogao Caves tickets, temperature tips)

## Example 3: Western Sichuan Mini Loop 5-Day Road Trip / 川西小环线 5 日自驾

**User input**: Plan a Western Sichuan mini loop starting from Chengdu, around 5 days, not too much driving. / 规划一个成都出发的川西小环线，5 天左右，不要开太久。

**Design approach / 设计思路**：
- Start/End: Chengdu (closed loop) / 起点/终点：成都（闭合环线）
- Total: ~900–1000km / 总里程约 900-1000km
- Altitude changes dramatically (500m → 4000m) with dense curves — daily driving relaxed to 3–5 hours / 川西海拔变化剧烈（500m → 4000m），弯道密集，驾驶时间需额外放宽至 3-5 小时/天
- 5-day moderate pace: snow mountains and grasslands without rushing / 5 天节奏适中，既能看雪山草原，又不会太赶

**Route / 路线节点**：

| Day | Route / 路线 | Mileage / 里程 | Duration / 时长 |
|-----|-------------|---------------|-----------------|
| Day 1 | Chengdu → Dujiangyan → Mount Siguniang / 成都 → 都江堰 → 四姑娘山 | 200km | 4.0h |
| Day 2 | Mount Siguniang → Danba Jiaju Tibetan Village / 四姑娘山 → 丹巴甲居藏寨 | 120km | 3.0h |
| Day 3 | Danba → Tagong Grassland → Xinduqiao / 丹巴 → 塔公草原 → 新都桥 | 150km | 3.5h |
| Day 4 | Xinduqiao → Kangding → Hailuogou / 新都桥 → 康定 → 海螺沟 | 180km | 4.0h |
| Day 5 | Hailuogou → Ya'an → Chengdu / 海螺沟 → 雅安 → 成都 | 280km | 4.0h |

**Core map nodes (10) / 核心地图节点（10 个）**：
Chengdu City, Dujiangyan, Mount Siguniang, Danba Jiaju Tibetan Village, Tagong Grassland, Xinduqiao, Kangding City, Hailuogou, Ya'an City, Chengdu City

**Output / 输出结果**：
- Personal map QR code (closed loop)
- Markdown itinerary guide (with altitude sickness prevention and mountain driving tips)

**Special notes / 特别提示**：
- Rapid altitude changes in Western Sichuan — avoid strenuous exercise at high altitude; prepare Rhodiola or oxygen canisters / 川西海拔变化大，初到高原避免剧烈运动，备好红景天或氧气瓶
- Day 2–4 cross multiple high-altitude passes consecutively — take extra rest stops / 连续翻越多个高海拔垭口，建议途中多休息
- Mid-to-late October is the best photography season at Xinduqiao, but watch for snow closures at Zheduo Mountain / 秋季（10 月中下旬）新都桥摄影最佳，但需留意折多山降雪封路
